# Strategy

赛道地图和策略决策目录。

- `track_map.*`：维护学习到的 RAM 路段事件表，并按里程查询当前/下一事件。
- `track_strategy.*`：根据当前循迹状态、IMU、编码器和地图事件输出车辆模式、目标速度和转向限幅。

第一圈学习逻辑是启发式实现，后续可根据实际赛道标志和传感器稳定性继续细化阈值。

丢线搜索会使用最后一次有效循迹误差决定搜索方向，避免丢线时无效大误差导致车辆固定向同一侧搜索。

地图学习和预测调速分别受 `CAR_ENABLE_TRACK_MAP_LEARNING`、`CAR_ENABLE_MAP_PREDICTION` 控制，便于先单独验证基础循迹和速度控制。地图/遥测持久化由上位机保存，MCU 端不擦写内部 Flash。
